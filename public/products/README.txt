FOTO PRODUK PER KATEGORI
========================

File .svg di folder ini adalah placeholder berdesain untuk tiap kategori.

Cara mengganti dengan foto ASLI:
1. Taruh file gambarmu di folder ini, contoh: minuman.jpg
2. Buka  common/libs/category-images.ts
3. Ubah path kategori itu, contoh:  Minuman: "/products/minuman.jpg"

Kalau file tidak ditemukan, kartu produk otomatis menampilkan ikon
yang cocok dengan nama produk (mis. 💧 untuk Air Mineral).
