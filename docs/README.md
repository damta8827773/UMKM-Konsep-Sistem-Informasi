## Ringkasan Sistem
**Warung Analytics** adalah sistem informasi penjualan & monitoring stok untuk **UMKM
warung kelontong**, terdiri dari:
- **Sisi Pembeli (Storefront):** katalog 1000+ produk, keranjang, checkout (pembayaran
  dummy), serta ulasan & komplain produk.
- **Sisi Admin (Dashboard):** manajemen stok, pemrosesan pesanan, KPI & grafik analitik
  (perputaran produk, belanja supplier), serta notifikasi ulasan/komplain pelanggan.
